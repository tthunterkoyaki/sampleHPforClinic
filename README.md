ASWは、元の固定利付債とは金利リスク特性が全く異なるため、独立して管理することは合理的です。しかし、ASWはカスタマイズされた複合取引であり、為替ヘッジ付き外債指数のようには広く配信されている標準指数が存在しません。

パフォーマンス要因分解（アロケーション効果と銘柄選択効果）を行うためには、市場の中立的なリターンを示すベンチマークが必要です。以下に、ASWのカスタムベンチマークを作成する具体的なアプローチと、代替案をご提案します。

### 1\. ASWの収益構造の理解

ASW（固定債を変動化する場合）は、投資家から見ると「合成変動利付債（シンセティックFRN）」となります。そのトータルリターンは以下の構造を持ちます。

```
ASWのトータルリターン ＝ キャリー収益 ＋ 価格変動
```

  * **キャリー収益** ＝ ① 基準変動金利 ＋ ② ASWスプレッド
      * ASWスプレッドは、原債券の信用リスクや流動性リスクの対価です。
  * **価格変動** ＝ 主に③ ASWスプレッドの変動による影響
      * 金利リスクはスワップで概ねヘッジされますが、市場の信用環境変化によりASWスプレッドが拡大・縮小すると、価格が変動します。

理想的なベンチマークは、これら3つの要素全てを反映する必要があります。

### 2\. 推奨アプローチ：カスタム（合成）ベンチマークの構築

精度と管理コストのバランスに応じて、3つの主要なアプローチが考えられます。

#### アプローチ1：静的スプレッド・モデル（簡便法）

管理が容易で、キャリー収益（①＋②）のみに着目する方法です。

  * **計算式:** `[基準変動金利指数] ＋ [固定スプレッド]`

##### 構築ステップ

1.  **基準変動金利の特定**:
      * 国内債ASWおよび為替ヘッジ後の外債ASW（円ベース）では、通常**TONA**（無担保コール翌日物金利）が基準となります。
2.  **固定スプレッドの設定**:
      * **推奨**: 評価期間の**期初時点**における、自行ASWポートフォリオの**加重平均ASWスプレッド**を計算し、これを固定値として設定します。（例：TONA＋30bps）

##### 評価と留意点

  * **メリット**: 計算が容易で、外部データが不要です。
  * **留意点（重要）**: 期間中の市場スプレッド変動による価格影響（③）が反映されません。例えば、市場環境悪化でスプレッドが拡大（価格下落）した場合、ポートフォリオは損失を被りますが、このベンチマークは影響を受けません。結果として、市場要因による損失が、銘柄選択の失敗として誤認されるリスクがあります。

#### アプローチ2：動的スプレッド・モデル（精緻法・推奨）

市場スプレッドの変動（③）までを反映し、トータルリターンを再現する方法です。

  * **計算式（概念）:**
    `[キャリー収益 (基準変動金利 + 市場ASWスプレッド)] － [ASWスプレッド変化幅 × スプレッド・デュレーション]`
    ※後段がスプレッド変動による価格影響（キャピタルゲイン/ロス）を示します。

##### 構築ステップ

1.  **ユニバースの定義**: 対象とする債券の属性（格付、年限）を定義します。（例：「国内社債 A格 5年」）
2.  **市場ASWスプレッド・データの取得**: 定義したユニバースの代表的なASWスプレッドの時系列データを、情報ベンダー（Bloomberg, Refinitiv等）や証券会社から取得します。
      * *外債（H有）の場合*: 通貨スワップのコスト（ベーシス）を含む「円転後のASWスプレッド」データが必要です。
3.  **計算**: データに基づき、キャリー収益と価格変動影響を合算して日次リターンを計算します。

##### 評価と留意点

  * **メリット**: 市場環境の変化を正確に反映するため、純粋な運用担当者の手腕（銘柄選択効果）を抽出できます。
  * **デメリット**: データの取得コストと、スプレッド・デュレーション（通常、原債の残存年限に近い値）の管理が必要です。

#### アプローチ3：インデックス・レプリケーション・モデル（最も精緻だが高負荷）

ASWの取引構造（債券購入＋金利スワップ）そのものを模倣する方法です。

  * **計算式:** `[原債の固定債インデックスのリターン] ＋ [金利スワップの理論的リターン（評価損益）]`

##### 評価と留意点

  * **メリット**: 理論的に最も正確です。
  * **デメリット**: 金利スワップの日次評価損益を計算するには、毎日の金利カーブデータと高度な金融工学計算が必要となり、負荷が非常に高いです。指数計算会社や運用会社への委託が現実的です。

### 3\. 代替アプローチ（簡便法）

#### 3.1. 変動利付債（FRN）指数をプロキシ（代理）として使用

ASWは合成FRNであるため、実際に市場で取引されているFRNの指数（例：変動利付国債指数）を代替ベンチマークとする方法です。

  * **留意点**: FRN市場とASW市場では需給が異なり、スプレッドの動きが一致しない場合があります（ベーシスリスク）。また、指数の構成銘柄が自行のポートフォリオと乖離する可能性があります。

### 4\. ベンチマーク以外の評価手法

伝統的なベンチマーク比較に加え、以下の視点も有効です。

#### 4.1. ファクター・ベースの要因分解

ポートフォリオのリターンを、指数との比較ではなく、共通の「リスク・ファクター」に分解する方法です。

ASWの場合、リターンを「短期金利ファクター」と「クレジットスプレッド・ファクター」に分解します。

  * **評価の視点**: 「金利リスク（デュレーション）は意図通り抑制できているか」「スプレッドを取ることでリターンに寄与できているか」を確認できます。これにより、「今期はスプレッドが縮小したためリターンが押し上げられた」といった、より本質的な分析が可能になります。

#### 4.2. ALM（負債）ベンチマーク

銀行の運用においては、ASWは変動金利負債（預金など）に対応するために保有されることがあります。この場合、負債の調達コスト（＋目標利鞘）をベンチマークとし、それを安定的に上回っているかを評価する視点も重要です。

### まとめと推奨アクション

パフォーマンス要因分解を目指す場合、市場の実態を反映することが重要です。

1.  **第一推奨（アプローチ2）**: まずは取引先証券会社や情報ベンダーに、対象ユニバースの「市場ASWスプレッド」データの提供可否を相談してください。可能であれば、動的スプレッド・モデルの構築を目指します。
2.  **第二推奨（アプローチ1）**: データ入手が困難な場合、自行ポートフォリオの「期初加重平均スプレッド」を用いた静的スプレッド・モデルを採用します。ただし、この方法はスプレッド変動リスクを捉えられない限界を理解しておく必要があります。
3.  **補完的分析**: 可能であれば、ファクター・ベースの要因分解を併用し、リスク特性が意図通りになっているかを確認することを推奨します。

### 1\. 国内債ASWと外国債券（H有）ASWのベンチマークは同じか？

結論から申し上げますと、国内債ASWと外国債券（為替ヘッジあり）ASWのベンチマークは**異なります**。

両者は最終的に「円の変動金利（例：TONA）＋スプレッド」という形で収益が得られる点では共通していますが、上乗せされる「**ASWスプレッド**」の決定要因と変動特性が根本的に異なるためです。

#### 収益構造とスプレッド要因の比較

ASWの収益は「基準変動金利 ＋ ASWスプレッド」で構成されます。

| 要因 | 国内債券 ASW | 外国債券（H有） ASW |
| :--- | :--- | :--- |
| **基準変動金利** | 円金利（例：TONA） | 円金利（例：TONA） |
| **① 信用・流動性リスク** | 国内発行体・国内市場のリスクプレミアム | 海外発行体・海外市場のリスクプレミアム |
| **② 通貨ベーシス** | （影響なし） | **影響大** |

#### 違いを生む主な要因

1.  **対象とする市場の違い（①）**:
    国内債と外国債では、発行体の信用力、市場の流動性、需給環境が異なります。同じ格付であっても、国内市場と海外市場では評価されるスプレッド水準や変動要因は異なります。

2.  **通貨ベーシスの影響（②、決定的差異）**:
    外国債券（H有）ASW特有の要因です。外貨建てのキャッシュフローを円に交換する際（クロス・カレンシー・スワップを使用）、「通貨ベーシス」と呼ばれるコスト（またはプレミアム）が発生します。
    これは通貨間の需給や金融市場の環境によって変動します。例えば、世界的にドルの需要が高まると、ドルを円に交換するコストが上昇し、外債ASWの収益性を圧迫（スプレッドが縮小）します。国内債ASWにはこのリスク要因はありません。

#### 結論

両者は異なるリスク・ファクター（特に通貨ベーシス）に晒されているため、パフォーマンスは連動しません。したがって、要因分解を適切に行うためには、**それぞれの市場実態を反映した別々のベンチマークを設定する必要があります。**

  * **国内債ASW**: 国内の対象ユニバース（例：国内社債A格）の市場ASWスプレッドを反映した指数。
  * **外債（H有）ASW**: 海外の対象ユニバースについて、通貨ベーシスを含む「円転後の市場ASWスプレッド」を反映した指数。

### 2\. インカム狙いの投資家へのリスク・パフォーマンス分析の訴求方法

「ASWはインカム狙いなので（時価）評価はしてほしくない」という意向は、満期保有（HTM）を前提とし、安定したキャッシュフローを重視する投資家（特に銀行勘定）には根強く存在します。

このような投資家に対し分析の必要性を訴求するには、短期的な時価変動（Mark-to-Market, MTM）の議論から焦点を移し、「**長期的なインカムの安定性と最大化**」のために分析がどう貢献するかを説明することが有効です。

#### 訴求アプローチ①：時価評価を「将来のインカムの健全性」を示す指標と位置付ける

ASWの価格変動は、主にASWスプレッドの変化（信用リスクや流動性リスクの市場評価の変化）によって引き起こされます。

  * **訴求ポイント**: 「時価（MTM）の下落は、市場がその資産の将来のインカムや元本償還に対する懸念を高めているシグナルです。時価評価は単なる価格の上下ではなく、保有資産の**健康診断**です。これを定期的に確認することで、ポートフォリオの信用力の悪化を早期に察知し、将来インカムが途絶える（デフォルト）リスクに備えることができます。」

#### 訴求アプローチ②：「機会費用（Opportunity Cost）」の観点から説明する

市場環境は常に変化します。時価評価とベンチマーク比較は、現状のポートフォリオが市場対比で有利か不利かを客観的に示します。

  * **訴求ポイント**: 「インカムを最大化するためには、常に市場と比較することが重要です。もし、保有資産よりも有利な投資機会（より高いスプレッド）が市場に存在するならば、現状維持は将来得られたはずのインカムを逃している（機会費用）ことになります。パフォーマンス分析は、こうした**インカム向上の機会を特定**するために不可欠です。」

#### 訴求アプローチ③：「流動性リスク（換金コスト）」の定量化として説明する

満期保有が前提であっても、予期せぬ資金繰りや規制対応により、売却を余儀なくされる可能性はあります。

  * **訴求ポイント**: 「万が一、市場環境が悪い（スプレッドが拡大している）時に売却が必要になった場合、MTMの損失が実現損失となります。リスク分析は、この潜在的な『**換金コスト**』を定量化し、流動性バッファとしての安全性を確認するために必要です。」
  * （特に銀行の場合）「最適化ツールでP/Lインパクト制約を考慮したように、将来のポートフォリオ組み換えの自由度を確保する上でも、潜在的な損失（含み損）の管理は重要です。」

#### 訴求アプローチ④：資本効率（RORA）の観点から説明する ※銀行向け

銀行の運用では、インカムの絶対額だけでなく、投下した資本（RWA）対比でのリターン（RORA）が重要です。

  * **訴求ポイント**: 「ASWは、スワップ取引に伴うカウンターパーティ・リスク等により、RWAを消費します。限られた自己資本をいかに効率的に活用して収益（インカム）を上げているかを評価することが、経営の健全性維持のために求められます。」

これらのアプローチにより、インカム重視の視点を尊重しつつも、分析の目的が中長期的なインカムの安定確保と最大化にあることを説明することが重要です。
