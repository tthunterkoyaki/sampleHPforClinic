### 1. `PRICE`関数を用いた債券リターンの計算方法

`PRICE`関数を使う場合、「現在、価格100円のパー債（クーポン＝現在の利回り）に投資する」という仮定を置くのが合理的です。

ここでは**1年後の期待リターン**を例に、具体的な計算ステップを示します。

#### 前提（例：日本5年金利）
* **現在 ($t_0$)**:
    * 5年金利 ($Y_0$): 1.22%
    * 投資対象: 残存5年、クーポン(C)が1.22%の債券
    * 投資価格 ($P_0$): 100円
* **1年後 ($t_1$)**:
    * 5年金利の見通し ($Y_1$): 1.20%

#### 計算ステップ

期待トータルリターンは「キャピタルゲイン（価格変動）」と「インカムゲイン（クーポン収入）」の合計です。

**ステップ1：1年間のインカムゲイン ($I_1$) を計算する**
インカムゲインは、1年分のクーポン収入です。
$I_1$ = クーポンレート ($C$) = **1.22円**
(※投資価格100円に対するリターン)

**ステップ2：1年後の債券価格 ($P_1$) を計算する**
`PRICE`関数を使って、1年後の価格を計算します。

* 1年後、投資した債券は「残存期間4年」になっています。
* この「残存4年の債券」を、1年後の金利（利回り）で評価します。

ここで**重要な「仮定（割り切り）」**が発生します。手元にあるのは「1年後の**5年金利**の見通し ($Y_1$=1.20%)」であり、「1年後の**4年金利**」ではありません。
最も簡易的に計算するため、ここでは**「1年後の4年金利」も「1年後の5年金利見通し(1.20%)」で代用する**、という仮定を置きます。

`P_1 = PRICE(settlement, maturity, rate, yld, redemption, frequency)`
* `settlement` (受渡日): $t_1$ (1年後)
* `maturity` (償還日): $t_0 + 5$年 (投資時から5年後)
* `rate` (表面利率): 1.22% (投資時のクーポン$C$)
* `yld` (利回り): 1.20% (1年後の見通し金利 $Y_1$)
* `redemption` (償還価額): 100
* `frequency` (利払回数): 2 (年2回)

この関数で $P_1$ の価格（例：100.07円など）が計算されます。

**ステップ3：トータルリターン（1年）を計算する**
1年間のトータルリターンは、1年後の価格と得られたインカムの合計です。

* **1年後のトータルリターン (年率) = $\frac{(P_1 + I_1)}{P_0} - 1$**
* **計算式: `( (ステップ2で計算したP_1) + 1.22 ) / 100 - 1`**

---
**補足（2年後の場合）**
2年後のリターンを計算する場合、インカムは2年分 ($I_2 = 1.22 \times 2$)、価格 ($P_2$) は残存3年の債券を「2年後の5年金利見通し($Y_2$=1.40%)」で評価する、という計算になります。
ただし、1年目のクーポンを再投資しない前提での簡易計算（単利計算）となるため、厳密な年率複利リターンとはズレが生じます。最も計算がクリーンなのは、上記のような1年間のリターン計算となります。

### 2. ドル建てリターン（S&P500）の円建て換算

S&P500の見通しから計算したドル建てリターンを円建てリターンに換算する方法は、「為替ヘッジをしない場合」と「する場合」で異なります。

#### A. 為替ヘッジなし（Unhedged）の場合

円建てリターンは、**「ドル建て資産のリターン」**と**「為替（ドル円）の変動」**の両方の影響を受けます。

**ステップ1：ドル建てリターン ($R_{\text{USD}}$) の計算** (年率)
これはユーザー様が算出された方法です。
* $S_0$: 現在のS&P500指数
* $S_N$: N年後のS&P500見通し
* $D$: S&P500の配当利回り（年率）
* $N$: 投資年数 (1年、2年...)
* **$R_{\text{USD}} = \left[ \left( \frac{S_N}{S_0} \right)^{\frac{1}{N}} - 1 \right] + D$**
    (＝ 年率キャピタルゲイン ＋ 年率インカムゲイン)

**ステップ2：為替リターン ($R_{\text{FX}}$) の計算** (年率)
* $FX_0$: 現在のドル円レート (例: 150円)
* $FX_N$: N年後のドル円レート見通し
* **$R_{\text{FX}} = \left( \frac{FX_N}{FX_0} \right)^{\frac{1}{N}} - 1$**

**ステップ3：円建てリターン ($R_{\text{JPY}}$) の計算** (年率)
これら2つのリターンを「掛け合わせ」ます。

* **$R_{\text{JPY}} = (1 + R_{\text{USD}}) \times (1 + R_{\text{FX}}) - 1$**

**例:**
ドル建てリターンが+8% ($R_{\text{USD}}=0.08$)、為替リターンが+2% (円安、 $R_{\text{FX}}=0.02$) の場合：
$R_{\text{JPY}} = (1.08) \times (1.02) - 1 = 1.1016 - 1 = 0.1016$
→ **年率 +10.16%** となります。（リターンの単純な足し算 8%+2%=10% ではない点に注意です）

#### B. 為替ヘッジあり（Hedged）の場合

為替変動のリスク（$R_{\text{FX}}$）を負わない代わりに、「ヘッジコスト（調達コスト）」が発生します。

円建てリターンは、**「ドル建てリターン」から「ヘッジコスト」を差し引いて**計算します。

**ステップ1：ドル建てリターン ($R_{\text{USD}}$) の計算**
上記Aのステップ1と同じです。

**ステップ2：ヘッジコスト ($Cost_{\text{Hedge}}$) の計算** (年率)
ヘッジコストは、日米の短期金利差で決まります。
* $i_{\text{USD}}$: 米国の短期金利（**FFレート**の見通しが使えます）
* $i_{\text{JPY}}$: 日本の短期金利（例：TONA, OISなど。これは別途見通しが必要です）
* **$Cost_{\text{Hedge}} \approx i_{\text{USD}} - i_{\text{JPY}}$**

※1年後のリターンを計算する場合、ヘッジコストは「現在のコスト」と「1年後のコスト見通し」の**平均値**などを使うのが合理的です。

**ステップ3：円建てリターン ($R_{\text{JPY}}$) の計算** (年率)
* **$R_{\text{JPY}} = R_{\text{USD}} - Cost_{\text{Hedge}}$**

**例:**
ドル建てリターンが+8% ($R_{\text{USD}}=0.08$)、平均ヘッジコストが3.5% ($Cost_{\text{Hedge}}=0.035$) の場合：
$R_{\text{JPY}} = 0.08 - 0.035 = 0.045$
→ **年率 +4.5%** となります。
